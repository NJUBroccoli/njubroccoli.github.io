# 行测笔记及资料收集

在此分享我学习粉笔980行测课程所做的笔记。

## 言语理解与表达

### 1. 片段阅读

#### 1.1 中心理解题

>- 提问方式
>   + 这段文字主要/旨在/重在/意在/想要说明（论述、强调） 的是……
>   + 这段文字的主旨/主题/观点是……
>   + 对这段文字概括最恰当的一项是……
>   + 这段文字表达了作者……
>   + 从这段文字中我们可以看出作者的意图/态度是……
>   + 适合做这段文字关键词的是……
>   + ……

- 转折关系
    + 转折之后是重点
    + 典型格式：虽然但是、尽管可是、不过、然而、却、其实/事实上/实际上
    + 错误选项特征：转折前的内容、围绕例子的表述、无中生有

>- 主题词
>    + 定义：文段围绕的核心话题，文段要围绕其展开
>    + 判断方法：一般前有引入或后有解释说明，每句话都围绕的相同话题
>    + 理论要点：正确选项中需包含文段主题词
>    + 错误选项特征：非重点、无中生有、偷换概念（偷换、范围扩大或缩小）

- 因果关系
    + 结论是重点
    + 典型格式：因为所以、由于因此
    + 结论标志词：所以、因此、因而、故而、于是、可见、看来、故、导致、致使、使得、造成
    + 文段特征
        1. 结论句在结尾：多为文段中心句
        2. 结论句在开头/中间：后为进一步的解释说明，则结论句为中心句；后有其他关联关系，则需结合多种关联词共同分析
    + 错误选项特征：非重点、无中生有、偷换概念

>- 程度词
>    + 标志词：更、尤其、正是、特别是、真正、根本、最核心、最突出等
>    + 理论要点：程度词可提示重点位置

- 必要条件关系
    + **“只有……才”中间是重点**
    + 典型格式：只有才
    + 对策标志词：应该、应当、应、需要、要、必须+做法、通过、采取……手段/途径/措施/方式/方法/渠道才能……、前提、基础、保障、负有……的义务/……的必由之路/……是法门之一/是要领之一/势在必行
    + 文段特征
        1. 对策在结尾：提出问题+分析问题+解决问题
        2. 对策在中间：提出问题+解决问题+解释说明（意义效果）
        3. 对策在开头：对策+正反论证/原因论证
    + 反面论证
        - 典型格式：如果/倘若/一旦……+不好的效果
        - 具体应用：把前面的做法反过来，即为对策
    + 提示：文段只出现“问题”表述时，“解决问题”通常会作为正确答案出现在选项中；对策不万能
    + 错误选项特征：非重点（围绕问题的表述、围绕意义效果的表述）、无中生有、偷换概念

- 并列关系
    + **全面概括**
    + 文段特征（主要有三种）
        - 包含并列关联词及标点（此外、另外、同时、以及、“；”）
        - 句式相同或相近
        - 按照时间顺序展开
    + 错误选项特征：非重点、无中生有、偷换概念、**表述片面**

- 行文脉络
    + 理论要点：把握中心句及分述句的特质
    + 中心句特征
        - 形式特征：重点词提示
        - 内容特征：观点（一般情况下是明确的）
    + 分述句特征
        - “比如”、“例如”、“……就是例证”等例子表述
        - 数据资料
        - 正反论证
        - 原因解释
        - 并列分述
    + 选项陷阱总结：尾句内容并非全部是重点；非重点处选项可能会设陷阱

#### 1.2 细节判断题

>- 提问方式
>   + 以下对文段理解正确/不正确的是……
>   + 符合/不符合这段话意思的是……
>   + 从文段中可以得知/推出的是……
>   + ……

- 错误选项类型
    + 无中生有
    + 偷换概念（替换、混搭）
    + 偷换逻辑（强加因果、因果倒置）
    + 偷换时态
        1. 将来时（将要、立刻、趋势、以后）
        2. 完成时（已、已经、了、完成）
        3. 进行时（正在、在……中、着）

- 快速解题技巧
    1. 对比项
        + 标志词：A比B更……
        + A高于/优于B
        + 利大于弊
        + ……
    2. 相对绝对项
        + 绝对表述：一定、必定、都……
        + 相对表述：可能、也许、往往……
    3. 表述与实际不符

### 2. 语句表达

#### 2.1 语句排序题

>- 提问方式
>   + 将以上/以下几个句子重新排列，语序正确的是……

- 解题要点：从选项入手
    + 确定首句
        - 首句特征：下定义（……就是/是指）、背景引入（随着、近年来、在……大背景/环境下）
        - 非首句特征：指代词（这、那、他、其、该、此）、关联词后半部分（但、却、而且）
    + 确定捆绑集团
        - 指代词：这、那、他、其、该、此
        - 关联词：配套出现确定捆绑 or 单个出现分析句意
    + 确定顺序
        - 时间顺序
        - 逻辑顺序：观点+解释说明、A和B
    + 确定尾句
        - 因此、所以、看来、于是、这……引导的总结性尾句
        - 对策

#### 2.2 语句填空题

>- 提问方式
>   + 填入画横线部分最恰当的一句/项是……

- 横线在结尾
    + 总结前文
    + 提出对策

- 横线在开头
    + 需对后文进行概括

- 横线在中间
    + 注意与上下文的联系
    + 把握好主题词，保证与文段话题一致


#### 2.3 接语选择题

>- 提问方式
>   + 作者接下来最有可能讲述的是……

- 理论要点：重点关注文段最后一句话
- **干扰项特征：文段中已经论述过的内容**
- 理解要紧紧依托文段，不能无中生有

### 3. 逻辑填空

#### 3.1 词的辨析

- 词义侧重（辨析方法）
    + 第一种：用不一样的字组词
    + 第二种：整体进行搭配组词

- 固定搭配
    + 搭配一个词：瞻前顾后找准搭配对象
    + 搭配多个词
        - 横线处所填词语搭配由“和”、“及”、“与”引导的并列结构
        - 需与并列结构搭配恰当

- **程度轻重**
    + 区分词的程度轻重
        - 程度轻（例如：“大相径庭”是指相差很大、相差很多）
        - 程度重（例如：“截然不同”是指完全相反、完全不同）
    + 理论要点：所填词语的程度轻重与文段意思的程度轻重保持一致

- **感情色彩**
    + 区分词的感情色彩
        - 褒义（如“成果”）
        - 贬义（如“后果”）
        - 中性（如“结果”）
    + 理论要点：所填词语的感情色彩与文段的感情色彩保持一致

#### 3.2 语境分析

- 关联关系
    + 转折
        - 标志词：但是、然而、却等
        - 理论要点：前后语义相反、感情色彩相反
    + 因果
        - 标志词：因为……所以、由于……因此、导致、使得等
        - 理论要点：横线前后构成因果关系
    + **并列**
        - 标志词
            + 同义并列：顿号（、）、逗号（，）
            + 反义并列：不是……而是……、是……不是……、相反、反之、多一些……少一些……、要……不能……等
        - 理论要点：同义并列连接前后语义相近；反义并列连接前后语义相反相对
        - 句式相同或相近：结合前后语境分析语义相近还是相反

- 对应关系
    + 解释类对应
        - 题干特点：分句，______，分句
        - 有标志：标志词（即、就是、可谓、无异于、无疑是等）、标点（冒号：、破折号——）
    + 重点词句对应：指代词、主题词、形象表达
    + 找准对应关系，要基于对整个文段意思的理解与把握

### 4. 篇章阅读

- 常见题型：细节判断题、中心理解题、词句理解题、标题填入题、语句填空题、逻辑填空题
- **命题思维**
    + 重点考查原文重现的内容
    + 题目难度与阅读难度成反比
    + 题目顺序与文段顺序基本一致
    + 文段内容“雨露均沾”
- **解题技巧**
    + 明确阅读顺序：问题->文段
    + 利用题干关键词定位原文：第X段、关于……、全篇、特殊标点、专有名词
    + 合理调整做题顺序
        1. 第一优先级：语句填空题、逻辑填空题
        2. 第二优先级：容易在原文定位的关于某具体话题的细节判断题
        3. 第三优先级：考查通篇主旨的中心理解题
        4. 第四优先级：不易定位，涉及全篇的细节判断题
    + 依托重点提示把握文段结构：文章首尾段、段落首尾句、文段内部关联词、特殊标点符号、专有名词

## 判断推理

### 5. 图形推理

#### 5.1 位置规律

- 图形特征：元素组成相同
- 平移
  + 方向：直线（上下、左右、斜对角线）、绕圈（顺、逆时针）
  + 常见步数：恒定、递增
- 旋转
  + 方向：顺时针、逆时针
  + 角度：45度、60度、90度、180度等常见度数
- 翻转
  + 左右翻转：两图沿竖轴对称
  + 上下翻转：两图沿横轴对称
- 小技巧：相邻比较
  + 比较题干：比较题干相邻两幅图的区别
  + 比较选项：比较剩下选项的区别

#### 5.2 样式规律

- 图形特征：元素组成相似
- 遍历
  + 图形特征：元素组成相似，且相同元素重复出现
  + 解题方法：缺啥补啥
- 加减同异
  + 图形特征：元素组成相似，且相同线条重复出现
  + 相加、相减
  + 求异：去掉相同线条，保留不同线条
  + 求同：去掉不同线条，保留相同线条
  + 注意：位置+样式的复合考法
- 黑白运算
  + 图形特征：图形轮廓和分割区域相同，且黑块数量不成规律
  + 解题方法：按照对应位置进行“黑白”相加运算，将所得规律应用于所求图形
  + 小技巧：从问号处入手，边找边验证

#### 5.3 属性规律

- 图形特征：元素组成不相同、不相似
- 对称性
  + 轴对称（特征图：“等腰”图形）
  + 中心对称（特征图：平行四边形，"S", "N", "Z"及相关变形图等）
  + 轴对称+中心对称（特征图：有两条相互垂直的对称轴的图形）
- 曲直性
  + 特征图：出现明显的圆、弧等全曲线图，优先考虑曲直性
- 开闭性
  + 特征图：出现生活化或粗线条图形，考虑开闭性

#### 5.4 数量规律

- 图形特征：元素组成不同且无属性规律；数量特征明显
- 面数量
  + 特征图
    - 图形被分割、封闭面明显
    - 生活化或粗线条图形中明显有空白区域
  + 细化考法
    - 所有面的形状
    - 相同形状的面的个数（注意三角形）
    - 最大面的形状、属性
- 线数量
  + 特征图
    - 直线数：多边形、单一直线
    - 曲线数：曲线图形（全曲线图、圆、弧）
  + 一笔画：图形能“一笔”画成，需要同时满足以下两个条件：
    1. 线条之间全部连通
    2. 奇点数为0或2个
  + 多笔画：连通图笔画数=奇点数/2（任何图形的奇点数一定是偶数）
  + 常见笔画数特征图：五角星、“日”“田”及其变形图、圆和圆相切与相交、多端点图形
- 点数量
  + 特征图
    - 线条之间交叉明显（俗称“大树杈”）
    - 图形相切较多
  + 细化考法
    - 曲直交点
    - 切点
- 元素数量
  + 元素种类、元素个数、元素换算
  + 部分数
    - 线条连在一起为一部分
    - 特征图：出现生活化或粗线条图形

#### 5.5 空间重构

- 六面体
  + 相对面
    - 同行或同列相隔一个面
    - “Z”字形两端，且紧靠着“Z”字形中间那条线
    - 应用：相对面同时出现的选项——排除
  + 相邻面
    - 两个面看公共边：平面图中构成直角的两条边是同一条边，四个面连成一行/列，两条的两条边是同一条边
    - 三个面看公共点：相邻三个面的公共点是唯一的
    - 画边法步骤
      1. 结合选项，找一个特殊面的唯一点或唯一边作为起点
      2. 顺/逆时针方向描边标号（描同一个面）
      3. 选项与题干对应面不一致——排除
- 四面体
  + 展开图中构成一条直线的两条边折合之后是同一条边
  + 展开图为平行四边形时，两条短边折合之后是同一条边

#### 5.6 平面拼合

- 原则：图形只能通过上、下、左、右平移，不能旋转、翻转
- 解题方法
  + 方法一：用特殊线条（如唯一的横线、竖线或斜线）秒杀
  + 方法二：拼题干
    1. 找到面积相对较大的面，以其为基准图形
    2. 找到和基准图形平行且等长的边，进行拼合
  + 方法三：分割选项

#### 5.7 实际意义

- 题型特征：题干给出一组具有日常生活意义的图形
- 解题思维：考虑图形的实际意义，找图形的共同特征
- 常见考法
  + 人：人数、种类、行为、衣着等
  + 动物：头朝向、种类等
  + 物品：形状等

### 6. 类比推理

#### 6.1 语义关系

- 近义关系
  + 例：开心：高兴；言不由衷：口是心非
- 反义关系
  + 例：勤奋：懒惰；一丝不苟：粗枝大叶
- 比喻义
  + 例：月亮：小船
- 象征义：借助某物的具体形象，以表现某种抽象的概念、思想和情感
  + 例：玫瑰：爱情

#### 6.2 逻辑关系

- 全同关系：两个词语意义完全相同
  + 例：土豆：马铃薯；苏：江苏
- 并列关系
  + 矛盾关系：是指对立的两种情况，不存在第三种情况，二者是非此即彼的关系
    - 例：生：死；对称：非对称
  + 反对关系：是指在对立的两种情况之外，还存在其他情况，二者不是非此即彼的关系
    - 例：红色：蓝色；草莓：香蕉
- 包容关系
  + 种属关系：A是B的一种
    - 例：白菜：蔬菜
  + 组成关系：A是B的一个组成部分
    - 例：镜片：眼镜
- 交叉关系
  + 有的A是B且有的A不是B
  + 有的B是A且有的B不是A
  + 例：男士：司机
- 对应关系
  + 高频对应关系
    - 材料（例：面粉：面包）
    - 功能（例：卷尺：测量）
    - 属性（例：沙漠：干旱）
    - 时间顺序（例：招标：投标）
    - 因果（例：努力：成功）

#### 6.3 语法关系

- 技巧：造句子
- 主谓关系
  + 例：歌手：演唱
- 动宾关系
  + 例：提出：问题
- 主宾关系
  + 例：农民：水稻

#### 6.4 拆词

- 相同单字反复出现
  + 例：寒：寒冷：寒舍
- 成语被拆分
  + 例：大同：小异
- 两个词语之间没有明显逻辑关系
  + 例：成败：呼吸

#### 6.5 符号类比

- 先看相同/不同字符的位置
- 再看字母大小写
- 最后看数字奇偶性

### 7. 定义判断

- 解题步骤
  1. 看清提问方式，为“属于/符合”“不属于/不符合”定义等
  2. 识别有效信息——找准关键词、关键句
  3. 注意分析选项，当纠结采用对比的思维进行排除，最终选择更符合题干要求的选项

#### 7.1 读得准

- 方法一：关键词——主体、客体
  + 主体、客体错误的选项一定不符合定义
- 方法二：句式
  + 常见关键词：方式（通过、利用）、目的（以、达到）、原因（因为、由于）、结果（导致、从而）、大前提（基于、前提是、在……时段）

#### 7.2 小技巧——同构选项排除法/拆词

- 同构选项排除法
- 拆词：定义本身词汇通俗易懂，优先关注，选项纠结时，可成为重要破题点

### 8. 逻辑判断

#### 8.1 翻译推理

- 题型特征

  1. 题干和选项中均出现明显的逻辑关联词
  2. 提问方式为“可以推出”、“不能推出”等

- 解题思维：先翻译，再推理

  + 注：尽量不要通过理解语义来解题，会有偏差

- 翻译规则

  + 前推后：如果……那么……、若……则……、只要……就……、所有……都……、为了……一定/必须……、……是……的充分条件
  + 后推前：只有……才……、不……不……、除非……否则不……、……是……的基础/假设/前提/关键、……是……的必要条件/必不可少的条件

  - 易错点：谁是必要条件，谁放在箭头后；除非……否则……

- 推理规则

  + 逆否等价：`A->B = -B->-A`
  + 传递规则：`A->B, B->C => A->C`

- “且”和“或”

  + “且”关系：……和……、……并且……、既……又……、不仅……而且……、虽然……但是……
  + “或”关系：……或者……、或者……或者……、……和……至少有一个（当“或”关系为真时，否定一项可以推出另一项）
  + 德·摩根定律：`-(A且B) = -A或-B`，`-(A或B) = -A且-B`

- 推理形式

  + 提问方式：以下哪项中的推理形式/结构与题干中的推理形式/结构相同
  + 解题思维：不要考虑推理是否正确，只考虑与题干是否一致

#### 8.2 组合排列

- 题型特征：题干中给出一组对象和相关信息，要求把对象和信息进行匹配
- 常用方法：排除法
  + 何时用：题干信息确定
  + 如何用：读一句，排一句
- 代入法
  + 何时用：题干信息有真有假；提问为“补充以下哪个条件”；提问为“可能/不可能”
  + 如何用：把选项代入题干验证
- 推理技巧
  + 最大信息：以题干中出现次数最多的信息为推理起点
  + 假设法：题干没有明确推理起点，且代入复杂，考虑假设
- 辅助工具（列表格）
  + 涉及排序
  + 对象和信息比较多，借助列表推理
- 特殊题型（材料题）
  + 与非材料题的解题方法一致
  + 通过材料直接推出来的结论可以直接应用于所有题目

#### 8.3 逻辑论证

#### 削弱论证

- 解题步骤
  1. 看清问法（削弱、加强/前提）
  2. 找出题干的论点和论据
  3. 根据论点和论据话题是否一致，预设可削弱/加强的方式
  4. 根据预设匹配选项，并对比择优
- 削弱方式
  + 否定论点
    - 题干特征：只有论点，无论据；或论点、论据话题一致
    - 选项特征：与论点表述的意思相反
  + 拆桥
    - 题干特征：论点、论据话题不一致
    - 选项特征：论点与论据之间没有必然联系（去同存异）
    - 注：提问方式为“削弱论证”时，可优先考虑拆桥
  + 否定论据
    - 题干特征：题干存在论据并且找不到否定论点和拆桥的情况下，考虑否定论据；题干中出现支持方、反对方观点相反的情况，经常采用否定论据的方式来削弱
    - 选项特征：与论据表述的意思相反
  + 因果类
    - 因果倒置：将论点中的因果关系顺序颠倒
    - 他因削弱：在论点原因1的基础上，选项增加另一个同时存在的、能导致相同结果的原因2，削弱的是原因1的重要性或者唯一性

#### 加强论证

- 题型特征：提问中包含“支持”、“加强”、“前提/假设”等
- 加强方式
  + 补充论据
    - 题干特征：只有论点，无论据；论点、论据话题一致
    - 解释：说明论点成立的原因
    - 举例：证明论点成立的例子
  + 搭桥
    - 题干特征：提问方式为前提、假设、加强论证时，优先考虑搭桥；论点、论据话题不一致
    - 选项特征：论点与论据之间有关系（去同存异）
  + 必要条件
    - 题干特征：提问方式为前提、假设、加强论证，且无搭桥项
    - 判别方法：没它不行

## 数量关系

### 9. 数字推理

- 基础数列：直接做
- 特征数列
  - 多重数列
    - 特征：项数多（$n\ge7$）
    - 方法：先交叉再分组
  - 机械划分数列
    - 特征：有小数点、乘号等特殊符号
    - 方法：以特殊符号为分界线，先分开再组内
  - 分数数列
    - 特征：全部或大部分是分数
    - 方法：观察趋势
      - 递增或递减：先分开再一起
      - 不递增或递减：先反约分，再按递增、递减做
  - 作商数列
    - 特征：两两倍数关系明显
    - 方法：两两作商找规律
  - 幂次数列
    - 特征：本身或附近有幂次数
    - 方法
      - 普通幂次：直接找规律
      - 修正幂次：转化为普通幂次$\pm$修正项
- 非特征数列
  - 多级数列
    - 特征：无明显特征，变化趋势平缓
    - 方法：两两作差，一次不行作两次
  - 递推数列
    - 特征：无明显特征，非多级数列
    - 方法：圈仨数、找规律、做验证

### 10. 数学运算

#### 10.1 代入排除法

- 范围
  - 典型题：多位数、年龄、不定方程、余数
  - 看选项：选项为一组数
  - 剩两项：只剩两项时，代入一项即得答案
- 方法
  - 优先排除：尾数、奇偶、倍数
  - 直接代入：最值、好算

#### 10.2 倍数特性法

- 基础知识
  1. 当B、C为整数时，如果$A=B\times C$，则A能被B、C整除
  2. 口诀：3、9看各位数字之和，4看末两位，5看末位
  3. 因数分解：分解时必须互质
  4. 拆分：拆成两个数的和或差
- 余数型
  - 若$y=ax+b$，则$y-b$能被$a$整除
  - 若$y=ax-b$，则$y+b$能被$a$整除
  - 前提：$a、x$均为整数
- 比例型
  - 若$\frac{A}{B}=\frac{m}{n}$，则$A$是$m$的倍数，$B$是$n$的倍数
  - 前提：$A、B$均为整数，$\frac{m}{n}$是最简整数比

#### 10.3 方程法

- 普通方程：设未知数
  - 设小不设大（避免分数）
  - 设中间量（方便列式）
  - 求谁设谁（避免陷阱）
  - 出现比例：设份数
- 不定方程：代入排除
  - 奇偶特征：系数一奇一偶
  - 倍数特征：系数与常数有公因子
  - 尾数特征：系数尾数为5或0
  - 直接代入选项
- 不定方程组
  - 未知数一定是整数：消元
  - 未知数不一定是整数：特值法（一般赋0）

#### 10.4 工程问题

- 给完工时间型
  - 先赋总量（公倍数）
  - 再算效率=总量/时间
  - 根据工作过程列方程
- 给效率比例型
  - 先赋效率（满足比例即可）
  - 再算总量=效率$\times$时间
  - 根据工作过程列方程
- 给具体单位型
  - 设未知数，找等量关系列方程
- 牛吃草类型
  - 题型判定：排比句；有增长、有消耗
  - 核心公示：$Y=(N-X)T$

#### 10.5 行程问题

- 普通行程
  - 路程=速度$\times$时间（$s=v\times t$）
  - 火车过桥：路程=车长+桥长
  - 平均速度
    - 平均速度=总路程/总时间
    - 等距离平均速度=$\frac{2v_1\times v_2}{v_1+v_2}$
- 相对行程
  - 相遇、追及
    - 相遇：$s_和=v_和\times t_遇$
    - 追及：$s_差=v_差\times t_追$
  - 多次运动
    - 线形两端出发第$n$次相遇：$(2n-1)s=v_和\times t_n$
    - 环形第$n$次相遇：$n圈=v_和\times t_遇$
    - 环形第$n$次追及：$n圈=v_差\times t_追$
  - 顺水、逆水
    - 顺水：$s=(v_船+v_水)\times t_顺$
    - 逆水：$s=(v_船-v_水)\times t_拟$
- 比例行程
  - $s$一定，$v、t$成反比
  - $v$一定，$s、t$成正比
  - $t$一定，$s、v$成正比

#### 10.6 经济利润问题

- 经济利润问题
  + 基础经济
    - 公式
      + 利润 = 售价 - 进价
      + 利润率 = 利润 / 进价
      + 折扣 = 折后价 / 折前价
      + 总价 = 单价 * 个数
  + 分段计费
    - 水电费、出租车费、税费等
    - 分段计算、汇总求和
  + 函数最值
    - 识别：单价和数量此消彼长，求最大利润或收入
    - 方法：根据条件列函数，求出极值点代入计算

#### 10.7 溶液问题

- 溶液问题
  + 基础公式
    - 浓度 = 溶质质量 / 溶液质量
    - 溶质质量 = 溶液质量 * 浓度
    - 溶液质量 = 溶质质量 / 浓度
  + 混合溶液
    - 识别：已知两种或多种不同浓度溶液混合，求混合浓度
    - 解题思路
      + 方程法：抓住混合过程溶质质量不变，列式求解
      + 线段法：浓度差与溶液质量成反比
  + 溶液不变
    - 识别：倒出去又加满
    - 方法：根据剩余比例计算

#### 10.8 容斥原理

- 容斥原理问题：画韦恩图

#### 10.9 排列组合与概率问题

- 排列组合与概率问题
  + 排列组合
    - 概念
      + 分类用加法（要么……要么……）
      + 分布用乘法（既……又……）
    - 题型
      + 情况数少：枚举法（依照次序）
      + 必须相邻：捆绑法（先捆再排）
      + 不能相邻：插空法（先排再插）
    - 正难反易：总情况数-反面情况数
  + 概率
    - 给情况求概率：满足要求的情况数/总情况数
    - 给概率求概率：分类用加法，分布用乘法
    - 正难反易：1-反面情况概率

#### 10.10 高频几何问题

- 高频几何问题
  + 周长
    - 正方形：$4a$；长方形：$2(a+b)$
    - 圆形：$2\pi r$；弧长：$2\pi r\times\frac{n^\circ}{360^\circ}$
  + 面积
    - 正方形：$a^2$；长方形：$ab$
    - 三角形：$\frac{1}{2}ah$；圆形：$\pi r^2$；扇形$\pi r^2\times\frac{n^\circ}{360^\circ}$
    - 梯形：$\frac{1}{2}(a+b)h$；菱形：$对角线乘积/2$
  + 表面积
    - 正方体：$6a^2$；长方体：$2(ab+bc+ac)$
    - 圆柱体：$2\pi r^2+2\pi rh$；球体：$4\pi R^2$
  + 体积
    - 正方体：$a^3$；长方体：$abc$
    - 柱体：$Sh$；锥体$\frac{1}{3}Sh$；球体：$\frac{4}{3}\pi R^3$

## 资料分析

#### 11.1 速算技巧

- 截位直除
  + 截谁
    - 一步除法：建议只截分母
    - 多步计算：建议上下都截
  + 截几位
    - 选项差距大：截两位
      + 选项首位不同
      + 选项首位相同，次位差大于首位
    - 选项差距小，截三位
      + 选项首位相同且次位差小于等于首位
- 分数比较
  + 一大一小
    - 根据分子比大小
    - 分子大的分数大，分子小的分数小
  + 同大同小
    - 竖着直接除：注意数量级
    - 横着看倍数

#### 11.2 资料阅读

- 资料阅读
  + 文字资料
    - 标记段落主题词，与题干进行匹配
    - 注意相近词、时间、单位等
  + 图表资料
    - 表格资料：横纵标目、标题、单位、备注
    - 图形资料：标题、单位、图例
  + 综合资料：不同类型资料之间的关系、资料结构

#### 11.3 基期与现期

- 基本术语
  + 基期量与现期量：资料分析中常涉及两个量的比较，作为对比参照的时期称为基期，对应的量称为基期量；而相对于基期的时期称为现期，所对应的量称为现期量
  + 增长量与增长率
    - 增长量：用来表述基期量与现期量变化的绝对量
    - 增长率：用来表述基期量与现期量变化的相对量
  + 同比与环比
    - 同比：一般与上年同一时期相比较
    - 环比：与相邻的上一个时期相比较

- 基期量
  + 识别：求前面某个时期的量
  + 公式：基期量=现期量-增长量；基期量=现期量/(1+增长率r)
  + 速算：r大截位直除，r小化除为乘
  + 基期和差：先用现期量和正负排除再计算

- 现期量
  + 识别：求后面某个时期的量
  + 公式：现期量=基期量+增长量；现期量=基期量*(1+增长率r)
  + 速算：截位计算，特殊数字

#### 11.4 一般增长率

- 基本术语
  + 增长率与倍数
    - 增长率指比基数多出的比率；倍数指两数的直接比值
    - 若$A$是$B$的$n$倍，则$n=r+1$（$r$指$A$与$B$相比的增长率）
  + 成数与翻番
    - 成数：几成相当于十分之几
    - 翻番：翻$n$番位原来的$2^n$倍
  + 增幅、降幅与变化幅度
    - 增幅一般就是指增长率，有正有负；
    - 降幅指下降的幅度，降幅比较大小时，前提必须为下降
    - 变化幅度指增长或下降的绝对比率，变化幅度比较大小时用增幅（降幅）的绝对值

- 一般增长率
  + 计算
    - 题型识别：增长/下降+%，成，倍
    - 解题方法：给百分点，直接加减；无百分点，$r=增长量/基期量$
    - 速算技巧：截位直除
  + 比较
    - 题型识别：增长最快/慢；增长率最高/低
    - 解题方法：$现期量/基期量\ge2$，比较现期量/基期量；$现期量/基期量<2$，比较增长量/基期量
    - 速算技巧：分数比较法
  
#### 11.5 增长量

- 基本术语
  + 年均增长量 = （现期量 - 基期量） / 年份差

#### 11.6 比重

- 基本术语
  + 比重指部分在总体中所占的比率，有时也用贡献率、利润率等表述方式
  + 增长贡献率指部分增量在总体增量中所占的比例
  + 利润率特指利润在收入中的占比
- 现期比重、基期比重、两期比重

#### 11.7 平均数

- 现期平均数、基期平均数、两期平均数

#### 11.8 倍数

- 现期倍数、基期倍数
- 倍数问法
  + $A$是$B$的$n$倍：$n=A/B$
  + $A$比$B$增长（多）$r$倍：$A/B=r+1$

#### 11.9 特殊增长率

- 间隔增长率
  + 题型识别：中间隔一年求增长率
  + 公式：$r=r_1+r_2+r_1\times r_2$
- 年均增长率
  + 题型识别：年均增长最快、年均增速排序
  + 公式：$(1+r)^n=现期量/基期量$
  + 比较方法：比较“现期量/基期量”（$n$相同）
- 混合增长率
  + 题型识别：部分增速与整体增速之间的关系

#### 11.10 其他

- 简单计算
  + 直接找数：注意范围等表述陷阱
  + 简单加减
    - 选项与资料精度相同：尾数法
    - 选项与资料精度不同：估算
  + 排序问题：时间、单位、主体、顺序
